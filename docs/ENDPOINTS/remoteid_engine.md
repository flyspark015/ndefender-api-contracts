# RemoteID Engine Endpoints

Base: `/api/v1`

## Read Endpoints
- `GET /status`
- `GET /health`
- `GET /contacts`
- `GET /stats`
- `GET /replay/state`

## Write/Command Endpoints
- `POST /monitor/start`
- `POST /monitor/stop`
- `POST /replay/start`
- `POST /replay/stop`

See `docs/ALL_IN_ONE_API.md` for full schemas and examples.
