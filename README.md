# ndefender-api-contracts
N-Defender Unified API Contracts & Integration Spec

Single source of truth for N-Defender: REST + WebSocket API contracts, schemas, examples, and UI integration guides across Backend Aggregator, System Controller, RemoteID, AntSDR, ESP32, and Observability.

- WebSocket catalog: `docs/WEBSOCKET_EVENTS.md`
- OpenAPI: `docs/OPENAPI.yaml`
- Models: `docs/MODELS/`
- Endpoints: `docs/ENDPOINTS/`
- Schemas: `schemas/`
- Contract gaps: `docs/CONTRACT_GAPS.md`
